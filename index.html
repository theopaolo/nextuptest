1• Comprendre le besoin du client et celui des usagers

Avant toute chose, il est fondamental de clarifier le besoin,
c’est-à-dire le « quoi » et le « pourquoi », avant de définir la solution : le « comment ».
cPour cela, il faut formaliser un « cahier des charges ».

Ce cahier des charges se focalise sur trois aspects très importants :

les aspects techniques ;

les aspects fonctionnels ;

les aspects économiques.


Ces trois éléments vont influencer l’architecture du dispositif EDP.
